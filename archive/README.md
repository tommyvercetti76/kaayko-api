# 📦 Kaayko API Archive

This directory contains **obsolete, experimental, and development files** that are no longer part of the active codebase but are preserved for historical reference.

## Directory Structure

### 📁 `obsolete/`
Files that are no longer needed for production but kept for reference:

- **`test-scripts/`**: Various test files from development phase
- **`development-docs/`**: Outdated documentation from API development
- **`debug-tools/`**: Debug scripts and experimental code
- **`legacy-functions/`**: Old function implementations that have been replaced
- **`old-scripts/`**: Deployment and testing scripts no longer in use
- **`testing/`**: Old testing framework and files

## ⚠️ Important Notes

- **DO NOT** deploy any files from this archive
- These files are kept for historical context only
- Active development should focus on the main codebase
- If you need to reference old implementations, check these archives first
